# Resumes Directory

Place your resume files here for automatic upload during job applications.

## Supported Formats:
- PDF (.pdf) - Preferred
- Word Document (.doc, .docx)
- Plain Text (.txt)

## Naming Convention:
Recommended naming: `my_resume.pdf` or `resume.pdf`

## Profile Picture:
If a job application requires a profile picture upload, place your photo here as:
- `profile_picture.jpg`
- `profile_picture.png`
- `photo.jpg`

## Notes:
- The bot will automatically detect and upload resume files from this directory
- Keep your resume up to date with the skills listed in `my_resume.json`
- Ensure file permissions allow reading
