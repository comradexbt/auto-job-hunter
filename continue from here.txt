📌 WHAT THIS PROJECT IS
An autonomous system that:
1. Scrapes job boards for listings
2. Uses Google Gemini AI to extract job details & match against your resume
3. Generates personalized cover letters
4. Auto-fills and submits job applications via browser automation
5. Sends Telegram notifications for each application

🧠 Gemini AI (ai_processor.py) — FULLY INTEGRATED:
  • Task A: Extracts job_title, company, remote status, skills, salary from page HTML
  • Task B: Scores job vs resume match (0-100), applies only if >= 70%
  • Task C: Generates personalized ≤200 word cover letters
  • Retry logic with exponential backoff
  • Falls back to regex/template if API is down

🤖 Telegram Bot (bot_telegram.py):
  • Runs in background thread
  • Commands: /start, /stats, /status, /recent, /help
  • Sends alerts: "✅ Applied to Senior Dev at Google"

🌐 Browser Engine (browser_engine.py):
  • Persistent Playwright context (saves sessions — login ONCE manually)
  • Stealth mode to bypass Cloudflare/bot detection
  • Human-like scrolling & typing
  • Smart field auto-detection (by label, placeholder, name, id)
  • Multi-page form support (basic)

💾 Database (db_manager.py):
  • SQLite with applied_jobs table (title, company, url, status, date_applied)
  • Prevents duplicate applications
  • Daily & total statistics

🗂️ Config Files:
  • my_resume.json — Your name, email, LinkedIn, portfolio, GitHub
  • target_sites.json — 9 example job boards (LinkedIn, Indeed, RemoteOK, etc.)
